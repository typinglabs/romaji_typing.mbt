# 仕様

- 画面には、スタート画面、カウントダウン画面、タイピング画面、終了画面の4つがある
- どれも共通のレイアウトを仕様する
- 画面には、打鍵に関するデータと、タイピング文章、フッターの3つが表示される

共通仕様

- 打鍵に関するデータは、打鍵数、ミス数、残り時間を表示する
- タイピングする文章は、漢字かな混じり文とローマ字を表示する
- フッターには、そのセッションの最高記録、最低記録、スタートボタンを表示する

スタート画面

- 打鍵数、ミス数、残り時間は全て0
- タイピングする文章の部分は何も表示しない

カウントダウン画面

- スタートボタンをクリックするか、Enterでゲームを開始する
- カウントダウンは3秒。打鍵数、ミス数、残り時間の3つに、3 -> 2 -> 1を順番に表示する
- ゲームが始まったら、打鍵数は0、ミス数は0、残り時間も0

タイピング画面

- 打鍵数とミス数を、打鍵に応じて更新していく
- 漢字かな混じり文とローマ字を、打鍵に応じて更新していく
- ミスした場合は、次に入力するローマ字の文字を分かりやすくハイライトする
- プレイ中にESCを押すとゲームを中断し、スタート画面に戻る

終了画面

- 制限時間が0になったら終了する
- 終了したら、漢字かな混じり文を表示していた場所に`Time Out!!　お疲れさまでした`と表示する
- 終了したら、ローマ字の表示部分に苦手キー情報を表示する
- 苦手キー情報は、`苦手キー：a(7)　s(2)　h(1)　u(1)　y(1)`という形式で、間違えたキーと間違えた回数を、回数が多い順に上位5個まで表示する
